# Anpassa e-postmeddelanden med hjälp av beslut

I den här självstudiekursen kommer vi att utforska ett verkligt användningsfall för att skicka personaliserade e-posterbjudanden med Adobe Journey Optimizer Decisioning. Scenariot är enkelt - skicka riktade e-posterbjudanden till kunder baserat på deras profilattribut, som årsinkomst och lojalitetsstatus.
Under hela självstudiekursen går vi steg för steg - från att skapa profiler i Adobe Experience Platform, till att konfigurera e-postkanalen och slutligen bygga och köra kampanjen som automatiskt väljer det mest relevanta erbjudandet genom att fatta beslut.

## Krav för den här självstudien

* Tillgång till Experience Platform

* Grundläggande förståelse för Experience Platform koncept (profiler, målgrupper, datauppsättningar)

* Välbekant med Journey Optimizer
