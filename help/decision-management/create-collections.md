# Skapa samlingar

Lär dig hur du skapar samlingar för beslutshantering. Samlingar används för att hantera erbjudanden i logiska grupper och krävs för att bygga upp beslutsaktiviteter. Med dynamiska samlingar kan du gruppera erbjudanden baserat på taggar, medan statiska samlingar gör att du kan strukturera erbjudandena manuellt.
