# Beslutsaktivitet för innehåll på arbetsytan för resan

Läs om hur användarna kan utnyttja Adobe Journey Optimizer Decisioning för personaliserade innehållserbjudanden med hjälp av innehållsbeslutsaktiviteten på arbetsytan. Använd den här funktionen för att dirigera profiler baserat på hämtade erbjudanden eller skicka erbjudanden till externa system, vilket förbättrar kundresan och gör den mer effektiv och personaliserad.
