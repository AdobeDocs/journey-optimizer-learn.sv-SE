# Rapportera och övervaka

Upptäck hur Adobe Journey Optimizer rapportering hjälper er att enkelt spåra, analysera och agera på kundresor.

De här självstudiekurserna ger marknadsförarna verktyg för att övervaka prestanda i realtid, fördjupa sig i detaljerad kundresa och kanalinsikter, exportera data, skapa aviseringar och utnyttja avancerad analys för smartare beslut.
