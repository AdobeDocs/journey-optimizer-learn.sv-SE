# Introduktion till e-postbeslut

Upptäck hur beslut kan omvandla e-postpersonalisering för era kampanjer. Den här guiden fokuserar på att använda beslut för att säkerställa att alla mottagare får de mest relevanta och högprioriterade erbjudandena. I resebranschen kan till exempel beslut på ett intelligent sätt välja och rangordna erbjudanden baserat på kundprofiler, vilket maximerar engagemanget och intäkterna.

Processen innebär att skapa en samling erbjudanden, tilldela regler för behörighet och tillämpa rangordningsmetoder som AI-modeller, formler eller statiska prioritetspoäng. Marknadsförarna kan skräddarsy urvalsstrategin för att visa de bästa erbjudandena effektivt i e-postmeddelanden. Testning säkerställer att strategin ger korrekta resultat.

Detta kraftfulla verktyg förbättrar kundengagemanget och ger slagkraftiga affärsresultat. Titta på den inbäddade videon för en detaljerad genomgång.
